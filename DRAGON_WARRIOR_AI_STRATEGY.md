## 1. 目标与边界

这不是一份让 AI 按费用拍牌的简单优先级表，而是一套可直接实现的决策规格。

AI 应当做到：

1. 每一步行动前先搜索本回合斩杀。
2. 识别快攻、节奏、慢速三类局面，并切换进攻/控场倾向。
3. 正确维护“手中有龙”“上回合打过龙”“本回合打过火焰法术”等状态。
4. 正确安排火焰法术、喷发火山、受伤目标与烈火炙烤之间的顺序。
5. 不为触发效果盲目浪费牌；效果收益必须大于卡牌和节奏成本。
6. 对随机发现、黑暗之赐、随机武器和回溯做可解释的评分。
7. 输出决策日志，方便后续通过真人对局调整权重。

本版不要求：

- 神经网络训练或调用 DeepSeek。
- 预知牌库顺序、对手手牌或随机结果。
- 为所有其他套牌编写专用策略；未知对手使用通用威胁模型。

## 3. 机制事实：引擎必须先实现正确

策略层依赖以下事实。任何一项没实现，AI 的评分都会失真。

### 3.1 “手中有龙”

- 检查战吼结算前的手牌，当前打出的牌已经离开手牌。
- 因此打出手中最后一条龙，不能让它自己满足“手中有龙”。
- 晦鳞巢母和先觉蜿变幼龙都需要**另一条**龙留在手里。
- AI 必须保留至少一张“龙锚点”，除非本回合能斩杀或该保留会造成明显节奏损失。

### 3.2 延系（Kindred）

- 如果玩家在**上一个自己的回合**使用过与该牌相同随从类型/法术派系的牌，则触发。
- 乘风浮龙是龙，所以要求上一个自己的回合使用过龙牌。
- 不是“本回合先打龙再打乘风浮龙”。
- 回合状态建议记录：`tribesPlayedLastOwnTurn` 与 `tribesPlayedThisTurn`。

### 3.3 火焰法术状态

本套牌火焰法术：

- 烈火炙烤
- 影焰晕染
- 灼热裂隙

喷发火山检查的是“本回合是否使用过火焰法术”，应记录 `playedFireSpellThisTurn`。

### 3.4 回溯（Rewind）

- 现场播报员第一次结算后，AI 可以保留时间线或回溯。
- 回溯会撤销该牌造成的全部游戏状态，再立即重打一次，不额外消耗法力和卡牌。
- 第二次随机结果可能与第一次相同。
- AI 只能根据已经看到的第一次结果决定是否回溯，不得偷看第二次随机数。

### 3.5 黑暗之赐

发现界面展示三个“随从+随机礼物”的组合，AI 选择整个组合，不能把某个礼物转移给另一张牌。

礼物包括：

| 黑暗之赐 | 效果 | 一般价值 |
|---|---|---:|
| 醒来吧，恐惧 | +3攻击、吸血 | 进攻/回血均强 |
| 裹得严实 | +4生命、嘲讽 | 对快攻很强 |
| 充分休息 | +2/+2、扰魔 | 稳定高价值 |
| 梦游者 | 冲锋 | 大攻击随从和终结战吼随从很强 |
| 鹰身人之爪 | 圣盾、风怒 | 高攻击随从极强 |
| 萦绕不去的恐惧 | 复生，复生时保留全部生命和附魔 | 高质量/战吼随从很强 |
| 短爪 | 费用-2、攻击-2 | 节奏强；低攻击随从可能不值 |
| 粗暴唤醒 | 战吼触发两次 | 高价值战吼的顶级礼物 |
| 活体梦魇 | 使用时召唤一个它的2/2复制 | 强站场，复制触发类效果尤其好 |
| 美梦 | +4/+5并置于牌库顶 | 资源强但延迟一抽，快攻局慎选 |

### 3.6 烈火炙烤的溢出伤害

- 只能选择已经受伤的随从。
- 当前伤害值为 `D`（初始8）。
- 若目标剩余生命为 `H` 且 `D > H`，目标死亡，并将一张伤害为 `D-H` 的烈火炙烤移回手牌。
- 若 `D == H`，刚好用尽伤害，不回手。
- 若 `D < H`，造成伤害但没有溢出，不回手。
- 回手版本必须保留当前剩余伤害，不能恢复为8。
- AI 要把它视为一份可跨多个受伤随从分配的伤害资源。

## 4. 套牌身份与总体计划

这是一套偏节奏/中速的龙战，不是纯控制战，也不是无脑打脸快攻。

### 4.1 第一阶段：1—3费建立节奏

目标优先级：

1. 场上留下随从。
2. 用低费牌补充龙和发现资源。
3. 保证手中仍有一条龙，开启龙巢守护者和晦鳞巢母。
4. 用武器和小法术处理对手关键随从，尽量让己方随从继续输出。

### 4.2 第二阶段：4—6费制造超模回合

核心是把减费与返费转换成同回合多动作：

- 4费先觉蜿变幼龙（手中另有龙）。
- 3费晦鳞巢母返2费，再接二费牌。
- 5费乘风浮龙（上回合打过龙）战吼打5、叠5甲。
- 火焰法术后启动6伤喷发火山。

AI 不应只追求“本回合打出费用最高的牌”，而应比较整条行动序列的总价值。

### 4.3 第三阶段：压血线与终结

- 场面领先：减少无必要交换，让随从与武器攻击英雄。
- 对手进入10—13血：每回合都搜索格罗玛什激怒斩杀。
- 格罗玛什 + 赤红深渊：最低9费，地标已在场时只需8费，10点冲锋伤害；赤红深渊还会使其本回合达到12攻（基础受伤10攻，再+2攻）。
- 格罗玛什 + 灼热裂隙：10费，先下格罗玛什再用裂隙，格罗玛什变10攻，英雄额外获得3攻；若英雄可攻击，总计通常13伤，另加已有武器攻击修正。
- 不要为了“留作斩杀”在必输的场面中死攥格罗玛什；必要时它可以冲锋解怪并留下高生命威胁。

## 5. 起手留牌

起手不是固定表；必须同时考虑先后手、对手速度和牌之间的条件。

### 5.1 通用优先级

**无条件优先留：**

1. 载蛋雏龙
2. 黑暗的龙骑士（起手已有另一条龙时）
3. 龙巢守护者（起手已有另一条龙时）
4. 晦鳞巢母（已有另一条龙，且至少还有一张1—2费可行动牌时）

**根据局面留：**

| 卡牌 | 留牌条件 |
|---|---|
| 永时困苦 | 对预计有1血/2血前期随从的快攻必留；慢速对局通常换掉 |
| 赤红深渊 | 有前期随从、格罗玛什组合需求，或需要连续补刀时留；没有随从曲线不单留 |
| 烈火炙烤 | 起手通常不留；确认对手会早出高生命大随从，且已有制造受伤手段时才考虑 |
| 鲜花商贩 | 有载蛋雏龙或后续龙曲线时留；对快攻没有一费动作时可作为2费站场留 |
| 影焰晕染 | 对快攻/节奏套牌留；慢速局有完整曲线才留 |
| 先行打击 | 对节奏套牌留；同时持有5费以上随从时价值明显上升 |
| 灼热裂隙 | 对铺场快攻必留；慢速对局换掉，除非已有喷发火山组合且曲线完整 |
| 喷发火山 | 有火焰法术且对手依赖场面时留；没有启动牌不单留 |
| 先觉蜿变幼龙 | 后手或慢速对局，有另一条龙并已有1—2费曲线时可留；否则换掉 |
| 乘风浮龙 | 只在慢速对局、已有早期曲线且有龙可在前一回合打出时保留 |
| 现场播报员 | 慢速对局且已有1—3费曲线可留；快攻局一般换掉 |
| 格罗玛什 | 一律不留，极慢对局也优先寻找前中期节奏 |

### 5.2 先手

- 更强调1费随从和连贯曲线。
- 不应保留两张相同的条件牌而没有触发龙。
- 示例好起手：载蛋雏龙 + 龙巢守护者 + 晦鳞巢母。
- 示例坏起手：两个晦鳞巢母 + 格罗玛什；虽然有龙，但前两回合完全空过。

### 5.3 后手

- 硬币允许保留一张强力3—4费节奏牌。
- 典型爆发：2费硬币晦鳞巢母，返2费后接龙巢守护者/鲜花商贩/解牌。
- 不要只因为有硬币就保留多个高费牌；硬币只能修复一个费用断点。

### 5.4 对快攻

留牌权重：低费站场 > 低费伤害 > 条件满足的返费牌 > 中期大龙。

优先组合：

- 载蛋雏龙
- 永时困苦
- 灼热裂隙
- 龙巢守护者 + 任意龙
- 火焰法术 + 喷发火山

### 5.5 对慢速

留牌权重：资源生成 > 超模曲线 > 小解牌。

优先组合：

- 黑暗的龙骑士 + 另一条龙
- 龙巢守护者 + 另一条龙
- 晦鳞巢母 + 另一条龙 + 二费牌
- 先觉蜿变幼龙 + 另一条龙（需已有早期动作）

## 6. 每张牌的详细决策规则

### 6.1 黑暗的龙骑士

**打出影响：** 1费站一个1/1，若手中另有龙则补充一张带黑暗之赐的龙，增加资源、提高后续龙条件稳定性。

**何时打：**

- 有龙时，一费优先级通常高于空放赤红深渊。
- 没有龙时，除非必须争夺场面或本回合需要耗尽法力，否则尽量等待触发。
- 手中只有一条龙且计划本回合打掉该龙时，先打龙骑士再打那条龙，避免丢失战吼。

**AI 评分修正：**

- 战吼成功：`+4.0`
- 无龙裸下：`-2.2`
- 对手有必须立即交换的1血目标：裸下惩罚减至`-0.5`

**起手：** 有另一条龙留；无龙通常换。

### 6.2 载蛋雏龙

**打出影响：** 最稳定的一费动作，本身是龙，既补牌又能为下一回合延系做标记。

**何时打：**

- 一费通常直接打。
- 若手中龙极少，它既能补龙，也能给下一回合乘风浮龙留下延系状态。
- 牌库接近疲劳、手牌接近10张时，才考虑延后。

**生成牌选择：** 此牌是随机获取而非发现，不需要 AI 选择；引擎只需加入随机三费及以下龙。

**起手：** 无条件留，重复两张时对快攻可都留，慢速通常只留一张并寻找更强曲线。

### 6.3 永时困苦

**打出影响：** 一张牌在“抽牌”与“召唤一费随从”之间切换。

**目标规则：**

- 若需要资源：优先点生命大于1、且点1后不会因其他效果死亡的低价值随从，触发抽牌。
- 若需要场面：优先击杀1血敌方随从，既解怪又召唤一费随从。
- 可以点己方高生命随从抽牌，但要扣除受伤代价；若能顺便开启格罗玛什，价值大幅上升。
- 不要为了抽牌点一个会因死亡触发/光环消失而立即死亡的目标。

**特殊组合：**

- 点未受伤格罗玛什：抽一张并开启10攻，通常是高价值动作。
- 点己方随从后接烈火炙烤通常没有意义，除非必须把伤害资源转移或清除己方被控制的随从。

**起手：** 快攻留，慢速换。

### 6.4 赤红深渊

**打出影响：** 三次“造成1点并+2攻”的可延后资源，不能把它只当成一费空场牌。

**目标优先级：**

1. 格罗玛什，立即形成12攻冲锋。
2. 当回合能攻击且不会因此轻易死亡的友方随从。
3. 需要先打伤才能被烈火炙烤选中的敌方大随从。
4. 1血敌方随从，仅在补刀收益高于给己方随从加攻时。

**使用限制：**

- 加攻只在目标存活时有价值；不要把己方1血随从点死，除非死亡收益明确。
- 若预计8费抽到/打出格罗玛什，尽量保留一次地标使用次数。
- 场面争夺紧张时，不应为了未来组合而拒绝当前的高价值交换。

**起手：** 有随从曲线时条件留。

### 6.5 烈火炙烤

**打出影响：** 低费硬解与伤害拆分工具，同时是火焰法术，可启动喷发火山。

**目标规则：**

- 只能打受伤随从，所以行动生成器不能创建非法目标。
- 优先击杀高威胁、嘲讽、持续成长或有强光环的随从。
- 若多个目标都需处理，优先选择剩余生命较低者，把更大的溢出伤害带回手中继续使用。
- 如果一个目标剩余1血、另一个剩余7血，用8伤版本先打1血，再用回手7伤打7血，可一费处理两个目标；行动搜索必须识别这种链。
- 若目标只受了1点但仍有9血，8伤无法击杀且不会回手，只在必须削弱它时使用。

**资源估值：**

- 回手剩余伤害的价值约为 `0.35 × 剩余伤害 + 0.8`，上限4分。
- 精确打光不回手不是错误；若因此击杀最高威胁，仍应优先。
- 为启动喷发火山，可以对合适的受伤随从先用小额回手版本，但不能无目标空放。

**起手：** 通常不留。

### 6.6 龙巢守护者

**打出影响：** 2/3站场并装备2/2武器，是高效率节奏牌；没龙时只是普通2/3。

**何时打：**

- 有龙时通常是最佳二费动作。
- 已有更强武器时，先计算覆盖损失：`旧武器攻击 × 旧耐久`。覆盖损失超过新武器总价值时，延后或先攻击。
- 当前武器还能攻击时，通常先用旧武器攻击再打守护者。
- 英雄血量危险时，避免用武器交换高攻击随从，除非能阻止更多伤害。

**起手：** 有另一条龙必留。

### 6.7 鲜花商贩

**打出影响：** 1/4耐打龙，每个回合末随机强化另一条友方龙。

**何时打：**

- 场上已有另一条龙时价值最高。
- 先打鲜花商贩再打另一条龙，可以保证回合末有强化目标。
- 如果场上有多条龙，AI 无法控制随机目标，应按平均收益估值。
- 需要为下回合乘风浮龙开启延系时，即使没有强化目标也可以打。

**起手：** 有龙曲线时留。

### 6.8 影焰晕染

**打出影响：** 2点灵活伤害、发现战士随从，并启动喷发火山。

**目标规则：**

- 能击杀关键2血随从时优先。
- 场面领先、对手无目标时可以打脸补资源，但先检查是否需要保留给火山或关键解牌。
- 不为了发现而把2伤浪费在己方关键随从上。

**发现评分：** 使用第9节“生成牌与黑暗之赐评分”，额外偏好能补当前费用曲线或提供立即防守的战士随从。

**起手：** 对快攻/节奏留，慢速条件留。

### 6.9 先行打击

**打出影响：** 2费3伤；手中有5费以上随从时额外抽一个随从。

**何时打：**

- 满足抽牌条件时，比普通3伤高约2.5—3分。
- 手中格罗玛什、乘风浮龙或未减费先觉幼龙可作为条件牌；“费用大于或等于5”按手牌中当时的当前费用判断，已降到4费的先觉幼龙不再满足。卡牌对象仍应同时保留基础费用和当前费用，便于规则与日志使用。
- 对3血关键随从优先；若能打脸并进入斩杀线，也可用于英雄。
- 不要为等抽牌条件而放任必须马上处理的威胁。

**起手：** 对节奏套牌留；有高费随从时更优。

### 6.10 灼热裂隙

**打出影响：** 全场随从1伤、英雄本回合+3攻、启动喷发火山、制造烈火炙烤目标、激活格罗玛什。

**使用前必须模拟：**

- 双方哪些随从会死亡。
- 己方会损失多少场攻和持续效果。
- 英雄是否能攻击；没有武器/其他攻击许可时，+3攻仍应按引擎规则允许英雄攻击一次。
- 是否能接火山或烈火炙烤。

**高价值场景：**

- 清掉至少两个1血敌方随从而己方损失较小。
- 格罗玛什在场：激怒并追加英雄3攻。
- 先裂隙打伤敌方大怪，再用烈火炙烤处理。
- 先裂隙，再启动6伤喷发火山。

**低价值场景：**

- 己方宽场明显领先，会杀死自己的关键随从。
- 英雄无法安全攻击，敌方也没有需要打伤/清除的随从。

**起手：** 对铺场快攻留。

### 6.11 晦鳞巢母

**打出影响：** 有龙时相当于净花1费得到4/3，但必须先拥有3点可用法力。

**行动排序：**

- 通常先打巢母返费，再用返还法力处理场面或铺二费牌。
- 不要先打掉手中最后一条龙导致巢母失去返费。
- 多张巢母在手时，其中一张可作为另一张的龙锚点；第一张打出后，第二张仍需检查手里是否还有其他龙。
- 返费不能超过本回合已消耗的空水晶；引擎按“复原”而非“临时获得”处理。

**AI 评分修正：**

- 成功返2费：`+4.5`
- 无触发裸下：`-2.5`，但必须争场时可接受。
- 返费后没有任何可用动作：收益仍存在于低净费用站场，但组合奖励降低1分。

**起手：** 有龙和低费后续时留。

### 6.12 喷发火山

**打出影响：** 地标可使用三次；普通3点敌方随机伤害，火焰启动后为6点。

**使用规则：**

- 伤害只随机分配给敌方角色/随从，具体目标池必须与游戏规则保持一致。
- 场上只有一个敌方目标时，全部伤害确定命中该目标，价值最高。
- 敌方有大量低价值小随从时，随机伤害容易浪费；用期望击杀数估值。
- 对手空场时若可伤害英雄，作为压血工具；若地点规则只命中随从，则不要生成空场使用动作。

**火焰组合：**

- 能打火焰法术时，不应先用3伤火山再打火焰法术，除非3伤已经足够完成目标且需要节省火焰牌。
- 启动成本低且火焰法术本身不亏时，6伤版本通常值得等待。
- 每次使用后按地标冷却规则，不能同回合连续使用同一座地标。

**起手：** 有火焰启动牌且对手靠场面取胜时留。

### 6.13 现场播报员

**打出影响：** 4费3/3龙，双方获得随机武器，我方武器+1/+1；随机结果可能帮助对手。

**打出前检查：**

- 我方现有武器是否会被覆盖；若会，先攻击。
- 对手是否已有强武器；新随机武器可能覆盖它，这是额外收益。
- 双方英雄血量；给对手高攻武器在低血时风险很高。

**第一次结果的回溯规则：**

计算：

`timelineValue = ownWeaponValue - opponentWeaponValue + replaceEnemyWeaponBonus - replaceOwnWeaponPenalty`

- `timelineValue >= 2.0`：保留。
- `timelineValue <= -1.0`：回溯。
- 中间区域：若我方领先且不想增加方差，保留非负结果；若我方落后，回溯寻找高上限。
- 不允许根据第二次未公开结果决定第一次是否回溯。

**武器价值建议：**

`weaponValue = attack × durability × 0.65 + immediateAttackValue + textEffectValue`

对手武器价值乘以血量风险系数：我方生命≤12时乘1.5，≤6时乘2.2。

**起手：** 慢速且已有早期曲线时条件留。

### 6.14 先觉蜿变幼龙

**打出影响：** 有另一条龙时4费5/8扰魔，是主要中期压场点。

**何时打：**

- 4费形态优先级极高，除非必须先解掉会滚雪球的威胁。
- 不要先打手中最后一条其他龙导致其恢复7费。
- 扰魔意味着不能被法术或英雄技能指定，但仍会受到群体/随机效果；AI 防守估值应提高，但不能视作无敌。
- 原价7费也可以作为厚实站场，但要和乘风浮龙、其他多动作回合比较。

**起手：** 慢速/后手，有另一条龙和低费曲线时留。

### 6.15 乘风浮龙

**打出影响：** 5点定向伤害、5甲、6/6本体；延系后5费，是中期最强节奏牌。

**规划规则：**

- 在预计下回合有5费且手中有乘风浮龙时，本回合打出一条龙的未来奖励约为4分。
- 若本回合有多个近似动作，优先选择“打龙”以开启下回合减费。
- 不能为了延系打出手中最后的龙锚点，导致其他牌连锁失效；要比较净收益。

**战吼目标：**

1. 能立即击杀的最高威胁随从。
2. 嘲讽随从，为场攻开路。
3. 敌方英雄，若能建立两回合斩杀或当前已无高价值目标。

**防守修正：** 我方生命≤12时，5甲价值提高50%；≤6时提高100%。

**起手：** 仅慢速、曲线完整且能可靠前一回合打龙时留。

### 6.16 格罗玛什·地狱咆哮

**打出影响：** 未受伤只有4攻，受伤为10攻；本套牌最重要的直接伤害来源。

**激怒手段：**

- 赤红深渊：最佳，额外+2攻，形成12攻冲锋。
- 永时困苦：1费激怒并抽牌，形成10攻。
- 灼热裂隙：激怒并给英雄+3攻，形成10攻格罗玛什 + 3英雄攻击。
- 让格罗玛什冲锋攻击随从：若存活，攻击结算后成为10攻，但本回合通常不能再次攻击；主要用于站场而非当回合追加伤害。

**决策规则：**

- 每回合先搜索全部激怒方式和攻击顺序。
- 对手生命在组合伤害范围内时，不得用激怒牌做非必要交换。
- 无法斩杀但对手下回合可能解掉格罗玛什时，直接当解牌使用需计算剩余场面。
- 对控制套牌可适当保留；对快攻若用它能稳定场面，不要贪斩杀。

**起手：** 永远换掉。

## 7. 行动顺序与组合模板

AI 应搜索完整行动序列，而非独立给每张牌打分。至少支持深度为6个动作的回合内搜索；动作包括出牌、地标、英雄攻击、随从攻击、英雄技能和结束回合。

### 7.1 通用顺序原则

1. 先做斩杀搜索，但不要立即执行，先确认随机分支是否必要。
2. 先进行确定性抽牌/发现，再决定剩余动作，除非先抽会爆牌。
3. 先攻击再覆盖武器。
4. 先打火焰法术，再使用需要火焰启动的喷发火山。
5. 先打晦鳞巢母返费，再花返还的法力。
6. 先用低伤害制造受伤状态，再用烈火炙烤。
7. 先做必要交换，再用随机伤害，或在随机伤害能显著减少交换成本时先随机伤害。
8. 鲜花商贩通常先于本回合其他龙打出，让回合末存在强化目标。
9. 所有攻击完成后再结束回合；除非攻击会触发对手有利效果或破坏战略目标。

### 7.2 关键模板

**模板A：晦鳞巢母返费**

`晦鳞巢母（手中留另一条龙） -> 返2费 -> 二费随从/解牌`

**模板B：火山满额伤害**

`影焰晕染/灼热裂隙/烈火炙烤 -> 喷发火山（6伤）`

**模板C：大怪分段处理**

`武器/地标/裂隙造成小伤 -> 烈火炙烤`

**模板D：炙烤连锁**

`8伤炙烤击杀低剩余生命目标 -> 回手剩余伤害 -> 继续击杀另一个受伤目标`

**模板E：格罗玛什斩杀**

- 地标已在场：`格罗玛什 -> 赤红深渊点格罗玛什 -> 12攻打脸`
- 9费：`格罗玛什 -> 永时困苦点格罗玛什 -> 10攻打脸并抽牌`
- 10费：`格罗玛什 -> 灼热裂隙 -> 格罗玛什10攻 + 英雄3攻`

**模板F：乘风浮龙规划**

`上一个自己的回合打任意龙 -> 下一回合5费乘风浮龙`

## 8. 攻击与交换策略

### 8.1 先判断角色

每回合计算：

`raceScore = ourTwoTurnDamage - opponentEffectiveHealth - opponentTwoTurnDamage + ourEffectiveHealth`

- `raceScore >= 8`：进攻模式，除关键威胁和嘲讽外尽量打脸。
- `raceScore <= -8`：防守模式，优先减少敌方场攻。
- 其余：节奏模式，做高效率交换并保留可持续场攻。

有效生命=`生命+护甲`。

### 8.2 必须优先处理的威胁

给敌方随从标记以下特征：

- 回合末/回合开始持续产生资源。
- 每次出牌、抽牌、召唤或攻击都会成长。
- 法术伤害、光环、风怒、吸血或关键部族引擎。
- 下回合可造成致命伤害。
- 嘲讽挡住我方斩杀。

威胁分建议：

`threat = attack × attackReadiness + engineValue + keywordValue + lethalContribution`

### 8.3 何时打脸

- 能本回合斩杀：全部资源服从斩杀。
- 能把对手压入已知下一回合格罗玛什斩杀线，且对手无法明显反杀：偏向打脸。
- 我方场面领先、对手没有成长引擎：偏向打脸，让对手负责交换。
- 对手有效生命很高且我方资源不足：优先取得场面和价值，不做低收益抢血。

### 8.4 武器交换

- 武器能保护两个以上己方随从继续攻击时，英雄承伤通常值得。
- 我方有效生命≤10时，每承受1点反击伤害的惩罚翻倍。
- 我方有效生命≤6且不能本回合斩杀时，除非目标本身会造成更多伤害，否则禁止用脸解高攻随从。

## 9. 发现与黑暗之赐评分

发现不是固定选最贵的牌。对每个候选计算：

`discoverScore = baseCardValue + curveFit + immediateImpact + synergy + giftValue - risk`

### 9.1 卡牌基础评分

- 当回合能打出并有效改变场面：`+3`
- 补当前缺失的下一回合曲线：`+2`
- 是龙且手中缺龙锚点：`+3`
- 能开启下一回合乘风浮龙延系：`+1.5`
- 提供定向伤害/嘲讽/回血等当前急需功能：`+2~5`
- 费用过高且预计三回合内不能打：`-2`
- 手牌9张：高费用纯资源牌`-3`，防止爆牌。

### 9.2 礼物情境评分

| 局面 | 高优先级礼物 |
|---|---|
| 对快攻、防守 | +4生命嘲讽、复生、+3攻吸血、+2/+2扰魔 |
| 抢血/斩杀 | 冲锋、圣盾风怒、+3攻吸血、费用-2 |
| 资源战 | 复生、双战吼、活体梦魇、美梦 |
| 当回合需要节奏 | 费用-2、活体梦魇、冲锋 |

### 9.3 特殊强组合

- 乘风浮龙 + 双战吼：两次打5、两次叠5甲，顶级选择。
- 乘风浮龙 + 活体梦魇/复生：高站场价值；复制是否复制战吼依照引擎规则。
- 大攻击龙 + 圣盾风怒：强终结威胁。
- 高攻击龙 + 冲锋：立即伤害。
- 先觉蜿变幼龙 + +4生命嘲讽：对快攻极强。
- 晦鳞巢母 + 双战吼：若规则允许重复复原法力，必须受“最多复原已消耗水晶”限制。
- 低攻击随从 + 短爪：攻击可能降到0，除非费用收益能立即救场，否则降权。
- 格罗玛什 + 短爪：攻击降低会影响激怒后的总攻，不能只看减费。

## 10. 局面评分函数

Claude Code 可从以下权重起步，后续通过对战日志调参。

```text
stateScore =
  1.00 * ourEffectiveHealth
- 1.15 * enemyEffectiveHealth
+ 1.35 * ourReadyAttack
+ 1.00 * ourSleepingAttack
+ 0.55 * ourMinionHealth
- 1.50 * enemyReadyAttack
- 0.90 * enemySleepingAttack
- 0.55 * enemyMinionHealth
+ 2.40 * handCardAdvantage
+ 0.35 * remainingDeckAdvantage
+ 0.65 * ourWeaponAttack * ourWeaponDurability
- 0.72 * enemyWeaponAttack * enemyWeaponDurability
+ synergyBonuses
- tacticalRisks
```

终局覆盖规则：

- 对手死亡：`+1,000,000`
- 我方死亡：`-1,000,000`
- 本回合可确定斩杀但行动序列没有选择斩杀：禁止该序列成为最优。

### 10.1 协同奖励

- 手中至少有1条龙：`+1.5`
- 手中有依赖龙的牌且有龙锚点：每张`+0.8`
- 上回合打过龙且手中有乘风浮龙：每张`+3.5`
- 喷发火山可用且本回合已打火焰法术：每座`+2.5`
- 格罗玛什在手且有低费自伤手段：`+2.0`
- 赤红深渊仍有耐久且格罗玛什在手：`+2.5`

### 10.2 风险惩罚

- 手牌超过8张：每张额外资源生成牌`-1.5`
- 本回合结束时剩余法力：每点`-0.45`，但不得因此乱交关键牌。
- 打掉最后一条龙导致两个以上条件牌失效：`-4.0`
- 用低价值动作覆盖高价值武器：按丢失武器价值全额扣分。
- 己方有效生命≤10时，让对手下回合已有场攻达到致命：`-1000`。

## 11. 决策算法建议

### 11.1 每个回合的主流程

```js
function chooseTurnPlan(state) {
  const legal = generateLegalActions(state);

  const lethalPlans = searchPlans(state, {
    objective: 'enemyHeroDead',
    includeRandom: true,
    maxActions: 8,
  });
  if (lethalPlans.hasCertainLethal) return lethalPlans.bestCertain;

  const role = classifyRole(state); // attack / tempo / defend
  const plans = beamSearch(state, {
    maxActions: 6,
    beamWidth: 40,
    evaluator: nextState => evaluateState(nextState, role),
  });

  return chooseRiskAdjustedPlan(plans, state, role);
}
```

### 11.2 随机效果

- 不读取未来随机种子。
- 小随机池可枚举所有结果并计算期望值。
- 大随机池用固定次数蒙特卡洛采样，例如每个动作32次。
- 领先时降低方差，落后时允许选择高上限路线。
- 回溯第一次结果后重新采样第二次结果，但第一次决策不得使用第二次信息。

### 11.3 动作剪枝

可以剪掉：

- 明确非法目标。
- 无效果的英雄技能/地标使用。
- 在不触发亡语等收益时，用伤害牌击杀己方随从。
- 未完成攻击就直接结束回合的方案。
- 覆盖武器前与覆盖武器后，仅保留“先攻击”通常更优的分支；若攻击有风险则两者都保留。

不能剪掉：

- 对己方随从造成1点的动作，因为可能激怒格罗玛什或触发永时困苦抽牌。
- 小伤害先打敌方大随从的动作，因为可能开启烈火炙烤。
- 看似低伤的火焰法术，因为可能把喷发火山从3伤提升到6伤。

## 15. 最低验收测试

以下测试全部通过，才算龙战 AI 第一版完成。

1. **最后一条龙测试**：手中只有晦鳞巢母自己，没有其他龙，打出时不得返2费。
2. **龙锚点测试**：手中有龙巢守护者、晦鳞巢母和一条龙，AI 不应先把唯一龙打掉再裸下两张条件牌。
3. **延系跨回合测试**：本回合打龙不能立即降低本回合乘风浮龙费用；下个自己的回合才生效。
4. **延系失效测试**：上回合没打龙，乘风浮龙保持8费。
5. **火山顺序测试**：同回合存在影焰晕染和可用火山时，AI 应比较“先火焰后6伤火山”，不能默认先点火山3伤。
6. **炙烤非法目标测试**：未受伤随从不能成为烈火炙烤目标。
7. **炙烤回手测试**：8伤击杀剩余3血的受伤随从，应回手5伤版本。
8. **炙烤连锁测试**：敌方有剩余1血和7血的两个受伤随从，8伤炙烤应能在一回合依次击杀两者。
9. **格罗玛什地标斩杀**：地标已在场、8费、对手12血时，应选择格罗玛什+赤红深渊并打脸斩杀。
10. **格罗玛什裂隙斩杀**：10费、对手13血、无嘲讽且英雄能攻击时，应识别格罗玛什+裂隙的13伤。
11. **武器覆盖测试**：已有武器可攻击时，打龙巢守护者/现场播报员前应优先攻击，除非攻击会导致我方死亡。
12. **回溯信息公平测试**：AI 决定是否回溯时不能访问第二次随机结果。
13. **黑暗之赐防守测试**：我方低血对快攻时，+4生命嘲讽的高质量随从应显著高于美梦慢资源。
14. **扰魔测试**：先觉蜿变幼龙不能成为敌方定向法术/英雄技能目标，但可受到群体和随机伤害。
15. **错过斩杀报警**：若存在确定斩杀而 AI 选择其他路线，测试必须失败并输出完整候选序列。
